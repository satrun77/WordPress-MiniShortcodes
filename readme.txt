== Description ==

This plugin contains mini "shortcodes".

(1) Display age based on a date.
(2) Display list of items.
(3) Display list of photos from Instagram.
(4) Display blog posts.

== Installation ==

See, http://codex.wordpress.org/Managing_Plugins#Manual_Plugin_Installation
